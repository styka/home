Analiza bezpieczeństwa (Security & OWASP Top 10)

OWASP Top 10 2021 — Stan aplikacji

1. Broken Access Control

✓ RBAC wdrożony (module.admin, module.shopping, ...): UserRole + Permission + RolePermission. Każda page.tsx sprawdza hasPermission(session, PERMISSIONS.MODULE). Self-lockout guard w access.ts.
⚠ Team ownership nie zawsze audytowany: Jeśli user edytuje /tasks/[projectId], czy sprawdzamy assertProjectAccess(projectId, userId)? Odpowiedź: TAK (każdy action ma access check), ale manualnie — brak middleware do automatyzacji.

2. Cryptographic Failures

✓ Komunikacja HTTPS (Render auto-HTTPS): Database URL via DIRECT_URL (szyfrowany). NextAuth JWT cookie secure + httpOnly.
⚠ API keys przechowywane w Config DB (plaintext): groq_api_key, brave_search_api_key widoczne w /admin/config. Rekomendacja: Szyfruj Config klucze via SecretBox (libsodium) lub HashiCorp Vault.

3. Injection

✓ SQL injection: NIE GROZI — Prisma ORM automatycznie parameteryzuje queries.
